# DuckDB analytics projects

This repository contains two independent dbt-DuckDB projects:

- [`nyc_cab_analytics/`](nyc_cab_analytics/) — NYC TLC trip analytics over HTTPS/S3 Parquet.
- [`overture_analytics/`](overture_analytics/) — configurable Overture Maps analytics over anonymous S3 GeoParquet.

Each project has its own README, profile, models, runner, outputs, and benchmarks. A virtual environment at the repository root can be shared by both project runners.
